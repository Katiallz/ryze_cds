📍 Sistema de CDS (Coordenadas Dinâmicas)

Sistema desenvolvido para facilitar a marcação e visualização de coordenadas dentro do servidor FiveM, ideal para criação de pontos, ajustes de mapas e organização de locais no GTA RP.

Permite selecionar posições com precisão diretamente no jogo, utilizando uma interface simples e prática.

⚙️ Como usar

Para iniciar o sistema, utilize o comando:

/cds3
🎮 Controles
Setas do teclado (↑ ↓ ← →) → Movimentar a bolinha no mapa

Page Up (PgUp) → Subir altura (eixo Z)

Page Down (PgDn) → Descer altura (eixo Z)

Tecla E → Copiar/pegar a coordenada atual

🚀 Funcionalidades

Seleção de coordenadas em tempo real

Ajuste de altura (Z) com precisão

Interface simples e intuitiva

Ideal para desenvolvimento de scripts e mapeamento
